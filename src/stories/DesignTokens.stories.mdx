# Design Tokens

Design tokens are the visual design atoms of the Rescale design system. They store visual design attributes such as colors, typography, spacing, and more as data that can be used across platforms and technologies.

## Animation

Duration and easing values for consistent motion.

### Durations

- `--rescale-duration-fast`: 150ms
- `--rescale-duration-base`: 250ms  
- `--rescale-duration-slow`: 350ms
- `--rescale-duration-slower`: 500ms

### Easing

- `--rescale-easing-ease`: ease
- `--rescale-easing-ease-in`: ease-in
- `--rescale-easing-ease-out`: ease-out
- `--rescale-easing-ease-in-out`: ease-in-out

## Usage

Design tokens can be used in CSS, styled-components, or directly in component props:

```css
/* CSS */
.button {
  background-color: var(--rescale-color-brand-blue);
  padding: var(--rescale-space-3) var(--rescale-space-4);
  border-radius: var(--rescale-radius-base);
  font-size: var(--rescale-font-size-base);
}
```

```jsx
// Styled Components
const Button = styled.button`
  background-color: var(--rescale-color-brand-blue);
  padding: var(--rescale-space-3) var(--rescale-space-4);
  border-radius: var(--rescale-radius-base);
`;
```

```jsx
// Inline styles
<div style={{ 
  color: 'var(--rescale-color-gray-700)',
  fontSize: 'var(--rescale-font-size-sm)' 
}}>
  Content
</div>
```
